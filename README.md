# Nextora - Smart Farming Landing Page Prototype

**Nextora** adalah proyek portofolio *Frontend Development* yang menggabungkan desain modern dengan kontrol interaktif. Proyek ini dibangun untuk menunjukkan fleksibilitas tata letak melalui fitur **Master Console**.

## 🚀 Fitur Utama
- **Master Console v1.0:** Panel kontrol real-time untuk mengubah desain tanpa bongkar kodingan.
- **Responsive Layout:** Tampilan optimal di desktop maupun perangkat mobile.
- **Data Logic:** Fitur Export/Import desain berbasis JSON.
- **Live Widgets:** Integrasi Jam Digital, Stopwatch, dan Ticker Data.

## 🛠️ Teknologi
- **HTML5 & CSS3** (Struktur & Animasi)
- **Vanilla JavaScript** (Logika Manipulasi DOM)
